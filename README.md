# Simple Feedback Form – Streamlit App

A small Streamlit app that collects basic feedback from users.

## Live App

👉 **Live demo:** https://app112.streamlit.app/

## What the App Does

The app shows a form where users enter:
- Name
- Age (10–100)
- Favorite subject (Math, Physics, CS, Other)
- Free-text feedback

On submit, it thanks the user by name and shows a summary of their inputs.
